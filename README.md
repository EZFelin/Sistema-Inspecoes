# Sistema de Inspeção de Equipamentos

O projeto Sistema de Inspeção de Equipamentos visa simular o processo de inspeção de equipamentos em uma empresa, permitindo o registro e acompanhamento das inspeções realizadas pelos responsáveis.

## Tecnologias que serão utilizadas:

Frontend
    1. HTML5
    2. CSS3
    3. JAVASCRIPT

Backend
    1. Python
    2. Flask

Database
    1.PostgreSQL

Ferramentas de desenvolvimento

    1. Git
    2. GitHub
    3. Visual Studio Code

## Funcionalidades Planejadas

    Cadastro de usuários
    Cadastro de equipamentos
    Registro de inspeções
    Histórico de inspeções realizadas
    Consulta de equipamentos cadastrados
    Armazenamento de dados em banco PostgreSQL
